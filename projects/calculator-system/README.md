# 🧮 Smart Calculator

> Python + Flet 기반 입력형 계산기

---

## 📌 Overview

숫자와 문자열을 구분하여 저장하고  
다양한 계산 기능을 제공하는 GUI 계산기입니다.

---

## 🚀 Features

- 숫자 / 문자 자동 구분 저장
- 사칙연산 (더하기, 빼기, 곱하기, 나누기)
- 평균 계산
- 최댓값 / 최솟값 계산
- Enter 키 기반 빠른 입력

---

## 🧠 Data Structure

- numbers : 숫자 리스트
- strings : 문자열 리스트

---

## ⚙️ How to Run

```bash
pip install flet
python main.py
