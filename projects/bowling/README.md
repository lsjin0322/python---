# 🎳 Bowling Scoreboard

> Python + Flet 기반 볼링 전광판 프로그램

---

## 📌 Overview

볼링 점수를 입력하면 전광판 형태로 표시하고  
스트라이크, 스페어 규칙을 적용하여 누적 점수를 계산하는 프로그램입니다.

## 🚀 Features

- 버튼 기반 점수 입력 (0~10)
- 스트라이크 / 스페어 자동 처리
- 프레임별 점수 표시
- 누적 점수 자동 계산
- 볼링 전광판 UI 구현

---

## 🧠 Logic

- 투구 기록(all_rolls) 기반 점수 계산
- 스트라이크 / 스페어 조건 분기 처리
- 프레임 단위 누적 점수 계산

---

## ⚙️ How to Run

```bash
pip install flet
python main.py
