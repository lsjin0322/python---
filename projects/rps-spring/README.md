# 🌸 Cherry Blossom Rock Paper Scissors

> Python + Flet 기반 GUI 가위바위보 게임

---

## 📌 Overview

사용자와 컴퓨터가 가위바위보를 진행하며  
먼저 3승을 달성하는 쪽이 승리하는 게임입니다.

## 🚀 Features

- 버튼 기반 가위바위보 선택
- 컴퓨터 랜덤 선택
- 승패 판정 로직
- 승리 횟수 실시간 표시
- 3승 도달 시 게임 종료
- 다시 시작 기능

---

## 🧠 Logic

- 딕셔너리 기반 승부 판정
- 상태 관리 (승리 횟수)
- 이벤트 기반 UI 처리

---

```bash
pip install flet
