# ✊✋✌ Rock Paper Scissors Game

> Python 기반 가위바위보 게임

---

## 📌 Overview

사용자와 컴퓨터가 가위바위보를 진행하며  
먼저 3승을 달성하는 쪽이 승리하는 게임입니다.

---

## 🚀 Features

- 사용자 입력 기반 선택
- 컴퓨터 랜덤 선택
- 승패 판정 시스템
- 승리 횟수 누적
- 3승 도달 시 종료

---

## 🧠 Logic

- 딕셔너리를 활용한 승패 판정
- 반복문 기반 게임 진행
- 조건문을 통한 종료 처리

---

## ⚙️ How to Run

```bash
python rps-game.py
